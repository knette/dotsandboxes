# Dots & Boxes

## How to Play
From Wikipedia: Starting with an empty grid of dots, two players take turns adding a single horizontal or vertical line between two dots. The player who completes the fourth side of a 1Ã—1 box earns one point and takes another turn. The game ends when no more lines can be placed. The winner is the player with the most points.
